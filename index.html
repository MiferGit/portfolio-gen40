<./Proyectos>

Pokédex

Una Pokédex realizada con HTML, CSS y React es una aplicación web interactiva que permite a los usuarios explorar información sobre los Pokémon de manera dinámica, ver sus detalles y realizar filtros, todo de manera rápida y eficiente gracias a la reactividad de React.

Booking App

BookingApp de hoteles desarrollada con HTML, CSS y React es una aplicación web moderna que facilita la búsqueda, filtrado y reserva de hoteles de manera interactiva y eficiente. Este BookingApp proporciona una experiencia fluida, donde los usuarios pueden explorar hoteles, aplicar filtros personalizados y completar su reserva de manera rápida y sencilla.

Rick and Morty App

La aplicación de Rick y Morty está construida con HTML, CSS y React y permite a los usuarios explorar el universo de la serie de manera interactiva. Los usuarios pueden filtrar personajes según su categorías y buscar personajes específicos para conocer más detalles sobre ellos. Además, se muestra la dimensión de cada personaje y si está vivo o muerto en el momento de la consulta. La aplicación se actualiza automáticamente con los datos más recientes, proporcionando una experiencia dinámica y fluida.

Weather App

Una app del clima desarrollada con React, HTML y CSS permite a los usuarios ver el estado del tiempo en tiempo real, incluyendo información como la temperatura, velocidad del viento y presión atmosférica. Utiliza React para gestionar el estado y actualizar la interfaz con datos dinámicos, obtenidos mediante una API de clima como OpenWeatherMap. El diseño de la app se estructura con HTML y se estiliza con CSS, garantizando una experiencia visual atractiva y responsiva.
